# Spatiotemporal_Volatility_Comparison
Code and replication files for a comparative study of spatiotemporal and network-based volatility models applied to financial time series.
This repository accompanies the paper
“Comparative Analysis of Spatiotemporal Volatility Models: An Empirical Study on Financial Network Series”
by Ariane N. Meli Chrisko, Jessie Li, Philipp Otto, and Wolfgang Schmid.
